QUICK START GUIDE
================================================================================

1. Environment Setup (5 minutes)
   $ cp .env.example .env.local
   $ nano .env.local
   → Set: EMAIL_SERVICE, JWT_SECRET, CSRF_SECRET, PASSWORD_SALT

2. Install Dependencies (3 minutes)
   $ npm install

3. Start Development (2 minutes)
   $ npm run dev
   → Open http://localhost:5173

4. Test OTP Flow (5 minutes)
   1. Click "Login"
   2. Enter test email
   3. Check console for OTP
   4. Enter OTP (default: check logs)
   5. Set password
   6. Verify login success

5. Deploy to Production (10 minutes)
   $ vercel deploy --prod
   OR use Netlify with environment variables from ENV_SETUP.md

ENVIRONMENT VARIABLES REQUIRED

For Local Development (.env.local):

  SENDGRID_API_KEY=SG.your-key (or EMAIL_USER/PASS for gmail)
  JWT_SECRET=your-random-32-character-string
  CSRF_SECRET=any-random-string
  PASSWORD_SALT=any-salt-string
  NODE_ENV=development

  SENDGRID_API_KEY=SG.your-production-key
  JWT_SECRET=production-secret-key
  CSRF_SECRET=production-csrf-secret
  PASSWORD_SALT=production-salt
  NODE_ENV=production
