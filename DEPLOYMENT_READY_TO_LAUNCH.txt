================================================================================
                            TROUBLESHOOTING
================================================================================

If deployment fails:

1. Git Push Error:
   - Ensure git is configured: git config user.name / user.email
   - Check internet connection
   - Verify GitHub access token if using HTTPS

2. Vercel Build Error:
   - Check Vercel deployment logs
   - Verify all environment variables are set
   - Ensure .gitignore excludes node_modules

3. Login Not Working:
   - Check SendGrid API key is valid
   - Verify DATABASE_URL is correct
   - Check email service logs

4. Database Connection Error:
   - Verify DATABASE_URL in Vercel env vars
   - Check Neon connection status
   - Verify connection pooling is enabled

For detailed troubleshooting, see: DEPLOYMENT_STEPS_FINAL.md
