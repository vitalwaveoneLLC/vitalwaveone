                        SUPPORT & TROUBLESHOOTING
================================================================================

If Git Push Fails:
  • Configure git user: git config --global user.name "Name"
  • Configure email: git config --global user.email "email@example.com"
  • Check internet connection
  • Verify GitHub access token

If Vercel Build Fails:
  • Check Vercel deployment logs (Deployments tab)
  • Verify all 8 environment variables are set
  • Ensure .gitignore excludes node_modules
  • Check build command in vercel.json

If OTP Emails Don't Send:
  • Verify SendGrid API key is valid
  • Check SendGrid quota/rate limits
  • Verify sender email is configured
  • Check email spam folder

If Login Fails:
  • Open browser console (F12)
  • Check error messages
  • Verify database connection
  • Check JWT_SECRET is set

If Database Connection Fails:
  • Verify DATABASE_URL in Vercel env vars
  • Check Neon connection status
  • Ensure connection pooling enabled
  • Test locally with same URL
