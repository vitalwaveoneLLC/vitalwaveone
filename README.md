# VitalWaveOne

A modern, full-stack SaaS platform built with React 19, Vite, Node.js, PostgreSQL, and Capacitor for mobile apps.

## Features

- **Frontend**: React 19 + Vite with HMR and ESLint
- **Backend**: Node.js API on Vercel
- **Database**: PostgreSQL (Neon)
- **Mobile**: Capacitor for iOS and Android
- **Authentication**: Clerk
- **Payments**: Stripe integration
- **Caching**: Upstash Redis
- **Deployment**: Vercel + GitHub Actions CI/CD

## Tech Stack

- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Mobile**: Capacitor (iOS/Android)
- **Backend**: Node.js + Express
- **Database**: PostgreSQL with Drizzle ORM
- **Auth**: Clerk
- **Payments**: Stripe
- **Testing**: Jest + Playwright
- **Hosting**: Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Build mobile app
npm run build:cap

# Run tests
npm test

# Run e2e tests
npm run test:e2e
```
